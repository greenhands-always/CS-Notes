# 基本配置
- 配置用户名和邮件地址
```bash
$ git config --global user.name "xxx"
$ git config --global user.email xxx@xxx.com
```
- 配置编辑器
```bash
$ git config --global core.editor [emacs/vim/...]
```
- 查看配置
```bash
git config --list
```
如果去掉 **--global** 参数只对当前仓库有效。
